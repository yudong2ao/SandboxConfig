# Windows 沙盒中文环境配置 (Windows Sandbox zh-CN)

这是一个轻量级的 Windows 沙盒 (`.wsb`) 配置文件。它的主要作用是在每次启动 Windows 沙盒时，自动在后台运行脚本，将沙盒系统的显示语言和输入法强制设置为**简体中文 (zh-CN)**。

由于 Windows 沙盒每次启动都会重置为一个纯净的初始环境，默认情况下经常是英文界面。使用此配置文件启动，可以完美解决语言障碍，省去每次手动调整语言的麻烦。

## ⚙️ 前置要求

* **系统版本**：Windows 10 或 Windows 11（需要是专业版、企业版或教育版）。
* **开启功能**：需在“启用或关闭 Windows 功能”中，勾选并安装了“Windows 沙盒” (Windows Sandbox)。
* **说明**：其它版本系统不支持Windows Sandbox功能。

## 🚀 基础使用方法

1. 下载本项目中的 `SandboxConfig.wsb` 文件到您的电脑上（建议存放在一个固定的文件夹，例如 `D:\Tools\`）。
2. 直接**双击**该文件。
3. Windows 沙盒会自动启动，并在进入桌面后自动执行后台脚本，为您呈现中文系统环境。

## 💡 进阶技巧：添加到“开始菜单”快速启动

为了像普通软件一样方便地随时调用，可以将配置文件建立一个快捷方式，并将其放入 Windows 的应用列表中：

1. **创建快捷方式**：找到下载好的 `SandboxConfig.wsb` 文件，右键点击它，选择“创建快捷方式”。
2. **重命名**：将生成的快捷方式重命名为您习惯的名字，例如：**`Windows沙盒`** 或 **`中文沙盒`**。
3. **打开开始菜单文件夹**：按下键盘上的 `Win + R` 键打开“运行”窗口，输入 `shell:programs` 并回车。这会打开当前用户的开始菜单程序文件夹。
4. **放入快捷方式**：将刚才重命名好的快捷方式（`Windows沙盒`）移动或复制到弹出的文件夹中。
5. **大功告成**：现在，您可以直接按下键盘上的 `Win` 键打开开始菜单，直接输入“Windows沙盒”就能快速搜索并启动它了！
